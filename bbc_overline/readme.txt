[color=purple][b]Compatibility[/b][/color]
SMF 2.0.x, SMF 2.1.x

[color=purple][b]Introduction[/b][/color]
This modification adds a BBCode button next to the "Strikethrough" button on the post page. You can use the [nobbc][o][/nobbc] BBCode to place a line above the selected text, as well as/as opposed to just Strike Through or Underline.

[color=purple][b]Installation Information[/b][/color]
The Package Manager should work in most cases, if you have problems installing please use the discussion thread as well as [url="http://docs.simplemachines.org/index.php?topic=402"]Manual Installation of Mods[/url]
